# Encyclopedia

Script creates encyclopedia site based on **config.json** file: generates title page with list of articles and article pages acordingly.

# Usage

    pip install -r requirements.txt
    python3 site_generator.py

It is possible to add new topics and articles to encyclopedia site. Information about new topics and articles should be included in **config.json** accordingly. Articles .md files should be placed in **articles** folder.
